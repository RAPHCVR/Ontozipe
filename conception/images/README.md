# Conception (PlantUML)

This folder gathers design diagrams for the frontend, backend, and core ontology.

- system-context.puml: global system context and external dependencies
- backend/modules.puml: NestJS modules, controllers, services, and dependencies
- backend/api-surface.puml: API controllers and base routes
- backend/auth-flow.puml: auth register/login profile sequence
- backend/ontology-flow.puml: ontology CRUD, PDF upload, and notification flow
- backend/llm-flow.puml: LLM ask flow with SSE and tools
- backend/notifications-flow.puml: notification read/write flow
- frontend/app-structure.puml: React app, providers, routes, and layout
- frontend/data-flow.puml: data access hooks, i18n, and API flow
- frontend/ontology-page.puml: ontology page component interactions
- frontend/guide-page.puml: guide page structure and data sources
- ontology/core-model.puml: core.ttl classes and object/data properties
- ontology/core-instances.puml: roles and notification verbs
